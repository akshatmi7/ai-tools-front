üß† What is This?

This is an AI-powered digit recognition app, where you can draw any digit (0-9) on a canvas, and the app will predict what number you drew. It works completely in your web browser using Machine Learning (ML) and AI models trained with your own data.

üöÄ How Does This Work?

  1. Drawing the Number
    You draw a digit on the HTML5 Canvas, which allows capturing your handwriting digitally.
  2. Capturing the Image
    The app converts your drawing into a grayscale image and resizes it to a smaller, standard size (e.g., 28x28 pixels) ‚Äî like how handwritten digits are processed in popular datasets like MNIST.
  3. Running AI Predictions
    The pre-trained AI Model, loaded using TensorFlow.js, processes your drawing and predicts the number you wrote.

‚úÖ TensorFlow.js

TensorFlow.js is a JavaScript library developed by Google that allows you to run Machine Learning models directly in your browser or in Node.js without needing any backend server. With TensorFlow.js, you can:

  • Train models in the browser.
  • Run pre-trained models (like ours) with no installations required.
  • Process images, audio, and text using AI.

‚úÖ Teachable Machine

Teachable Machine is a no-code web tool by Google that lets you create machine learning models using:

  • Images
  • Sounds
  • Poses

Steps we followed using Teachable Machine:

  1. Collected several images of hand-drawn digits by drawing them and taking screenshots.
  2. Uploaded those images to Teachable Machine.
  3. Assigned them to classes like 0, 1, 2, etc.
  4. Clicked "Train" to let Teachable Machine train a neural network using your images.
  5. Exported the model as a TensorFlow.js compatible model, including the model.json and weight files.

Now, the AI model remembers how digits look like based on your training images, and predicts new inputs.

üîç What is TensorFlow?

TensorFlow is an open-source machine learning framework created by Google. It is widely used to:

  • Train deep learning models
  • Build AI solutions like image classification, text generation, and voice recognition
  • Deploy models on servers, mobile apps, and now in the browser with TensorFlow.js

üéØ How Did I Train the Model?

  1. I first drew digits (0-9) several times on a drawing app and took screenshots of each digit.
  2. I uploaded these images to Teachable Machine, creating classes for each digit.
  3. Teachable Machine automatically built and trained a neural network model, adjusting it to learn from my examples.
  4. Once trained, I exported the model for TensorFlow.js, which gave me a model.json file and some binary weight files.
  5. I placed these files in my Angular app's assets and loaded them using TensorFlow.js.

üõ°Ô∏è Why Train a Custom Model Instead of Using MNIST?

Popular datasets like MNIST contain standard handwritten digits, but they may not match your unique drawing style. By training your own model:

  • The model gets better at recognizing your handwriting.
  • You can experiment with fewer training images.
  • You learn how AI adapts to custom data.
